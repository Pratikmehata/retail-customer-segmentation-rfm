## Retail Customer Segmentation using RFM & K-Means

### Problem Statement
Businesses need to understand customer purchasing behavior to improve retention and revenue. This project segments retail customers based on Recency, Frequency, and Monetary value.

### Approach
- Cleaned transactional retail data
- Created RFM features at customer level
- Applied K-Means clustering
- Validated clusters using Silhouette Score and PCA visualization
- Interpreted clusters into actionable business segments

### Technologies
Python, Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib

### Results
Identified four customer segments: VIP, Loyal, Regular, and At-Risk customers, enabling targeted marketing strategies.
